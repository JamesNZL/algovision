# Artwork Guidelines

## Via Sizes

| Size   | Pad              | Drill             |
| ------ | ---------------- | ----------------- |
| Large  | $0.7\,\text{mm}$ | $0.3\,\text{mm}$  |
| Medium | $0.6\,\text{mm}$ | $0.25\,\text{mm}$ |
| Small  | $0.5\,\text{mm}$ | $0.2\,\text{mm}$  |

> [!warning] JLC Capabilities
> 1 & 2 Layer PCB: 0.3mm(hole size) / 0.5mm(diameter)
